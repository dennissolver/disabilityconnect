# DisabilityConnect

AI-powered disability support service finder.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create .env file:
```
ANTHROPIC_API_KEY=your-key-here
```

3. Run locally:
```bash
npm run dev
```

4. Open: http://localhost:8888

## Deploy to Netlify

1. Push to GitHub
2. Connect to Netlify
3. Add ANTHROPIC_API_KEY environment variable
4. Deploy
